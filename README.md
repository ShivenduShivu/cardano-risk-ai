# Cardano Risk AI

AI-powered Cardano Risk Monitoring — cloud-first, Codespaces-ready.

## Quick start (Codespaces)
- Open this repo in GitHub Codespaces (Code → Open with Codespaces).
- It will install Python dependencies from backend/requirements.txt.
- Run backend: `uvicorn app.main:app --reload --host 0.0.0.0 --port 8000`
